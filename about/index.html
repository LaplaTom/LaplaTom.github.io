    • 描述：本项目针对校园内共享单车的管理，车辆停放，特殊时期、特殊路段的车辆管理等问题，建立一个校园车辆监测管理系统。该系统可以实时监控校园内车辆情况，一旦发现共享单车违规摆放、车辆违规进入管制区域、车辆乱停放等问题，该系统可即时反馈给校园相关管理人员或者车主，以实现车辆的规范管理。
    • 我的职责：本项目分为共享单车的识别、车辆综合特征的识别、车辆的入侵检测、车辆违规停放检测四个部分。其中我们通过收集校园内共享单车与行人的图片进行共享单车与行人的检测和特征识别，另外我们通过将 YOLOv3 算法与Deep Sort 算法想结合，以前后三帧作为多目标跟踪的相似度判断，最终实现行人与单车的检测框为中心的轨迹实现。并且以该轨迹为基础，通过人为交互设计违停区域，实现了车辆的入侵检测与违规停放功能。
    • 成果：1. 本项目在 NVIDIA Jetson TX1 机械上实现了每秒 24 帧的识别效率，达到了实时的效果、2. 本项目作为本人的本科毕设作品

  • 基于 web 输入的联机笔迹鉴别系统，福建厦门，2023.03-2023.04
    • 描述：本项目提出一种基于 web 输入的联机笔迹鉴别系统。使用者可以通过在 web 上的签字版上通过签五个自己的签名作为系统留底，之后的每一个签名与预留的五个签名进行对比，对比结果的判断率超过设定的阈值（默认为 0.6）即为判断正确。因为该联机系统为基于 web 实现数据的输入，因此本系统在平板、手机、PC 三段均可输入测试
    • 我的职责：本项目采 web 的 canvas 模块作为输入，获取到联机笔迹点，传到服务器后端转为图像，通过模型与系统保存的模型进行比较。后端由 flask 实现，签名验证模型由自己独自训练的模型来判断。
    • 成果：笔迹验证准确率达到 80+，项目即将开源在 Github 上。

  • 河南国家电网《表单关键信息提取》，福建厦门，2023.10-2023.12
    • 描述：导师横向项目。项目的目标为检测、识别、模板匹配、版面分析及矫正、关键信息提取及后处理河南电力系统当中的信息表单文本数据, 用于支撑相应应用场景下的脱机文本表单图片的文字处理相关任务。多个应用场景包括：文字识别、文本行检测、文本行识别、版面分析、关键信息提取及规则判断等。
    • 我的职责：本项目采用多模型整合而成，当我们拿到表单文档的时候，先通过 PyTorch 的文档矫正算法进行文档矫正，然后中科院 API 和自己基于 Paddle 训练的基于文本行进行 OCR 识别的模型同时进行文本行的检测和识别，将检测识别到的文本信息及位置通过 Paddle 的关键信息提取算法进行训练。
    • 成果：在 3000+ 测试图片上均取得不错的效果，可构建 api 或基于 docker 的部署，单张图片处理时间小于 1s。
